# Deep Learning Framework from Scratch

A lightweight, layer-oriented deep learning framework built from scratch using NumPy. This project implements the fundamental components of a neural network framework, including optimizers, layers, activation functions, loss functions, and a network manager.

## Overview

This framework provides a clean, educational implementation of core deep learning concepts. It follows a layer-oriented architecture where each layer implements `forward()` and `backward()` operations. The framework includes:

- Stochastic Gradient Descent (SGD) optimizer
- Fully Connected (Dense) layers
- ReLU activation function
- SoftMax activation function
- Cross-Entropy Loss
- Neural Network manager for training and testing
